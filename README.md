# Ex-cution-d-branch-e
Le projet "Système d’Exploitation Humain" est une activité débranchée de Terminale NSI visant à vulgariser le fonctionnement interne d'un OS par l'analogie physique. La classe devient un ordinateur où les élèves incarnent des processus (instances actives) exécutant des instructions simples (prendre, poser, dessiner) à partir d'une fiche programme.
